# Unow Contact Form Plugin

## Descripción
Este plugin permite a los usuarios enviar un formulario de contacto cuyos datos se almacenan en la base de datos. Los datos son accesibles desde el administrador de WordPress.

## Características
- Formulario de contacto con nombre, correo y mensaje.
- Guardado de datos en la base de datos.
- Listado de entradas en el administrador.

== Instalación ==
1. Sube el archivo `unow-contact-form.zip` desde el panel de WordPress.
2. Activa el plugin desde el menú de **Plugins**.

## Uso
1. Activa el plugin.
2. Usa el shortcode `[unow_contact_form]` para mostrar el formulario en una página o entrada.

== Aviso Importante ==
Al desinstalar este plugin, se eliminarán todos los datos almacenados en la base de datos asociados con el formulario de contacto. 
Recomiendo realizar una copia de seguridad antes de desinstalar
